# tic-tac-toe

- Project to learn Clojure, a functional Lisp on the JVM.
  - https://clojure.org/
- GUI made with cljfx, a JavaFX wrapper.
  - https://github.com/cljfx/cljfx

## Installation
- Download a Jar file from releases page.
